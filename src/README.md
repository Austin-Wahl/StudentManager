# Student Manager

Student Manager is a JavaFX application built for UTSA CS-2123 which illustrates the usage of HashTables and Priority Queues through a custom implementation.

## Technologies

- Java
- JavaFX

## Data Structures

The following data structures were built from the ground and do not use external dependencies except Arrays and ArrayLists

- PriorityQueue
- HashTable

## Additional Info

This application does now allow CREATE, UPDATE, or DELETE commands. It only reads data from the `/src/main/data.txt` file. This data is processed and turned into a Java StudentTableRecord object which is then used by the TableView and rendered to the screen.

Seed/test data can be generated by compiling the `DataGen` class located in `/src/main/java/utis/DataGen.java`. Compile with `javac DataGen.java` and run with `java DataGen`. The data is written to the file. Be sure to move the generated data.txt file to the `/src/main/` directory.
